1) (5,0 pontos) No contexto do desafio apresentado, prepare uma base de dados em Excel contendo:
a) No mínimo 2 variáveis qualitativas nominais e 2 variáveis qualitativas ordinais, com pelo menos 50 observações.
b) No mínimo 2 variáveis quantitativas discretas e 2 variáveis quantitativas contínuas, com pelo menos 50 observações.

2) (5,0 pontos) Utilizando a base de dados do item (1), montar a Tabela de Distribuição de Frequências para:
a) 2 variáveis quantitativas discretas. Na sequência, extraia, pelo menos, 2 insights das tabelas montadas.  
b) 2 variáveis quantitativas contínuas. Na sequência, extraia, pelo menos, 2 insights das tabelas montadas.

3) Ao término do Challenge Sprint 1, cada grupo deverá entregar:
- O arquivo com a base de dados em Excel solicitada no item 1 (formato .XLSX).
- O arquivo com as tabelas e os insights solicitados no item 2 (formato .PDF).
